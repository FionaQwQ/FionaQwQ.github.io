迪英加科技主要产品分析

Posted on 2020-05-11

一、产品理念：

AI辅助诊断，智能点亮病理

二、产品定位：

致力于深耕AI+病理的业务方向。通过完全自主研发的AI算法，开发具有多模块的AI辅助诊断工具，并以此为基础展开包含云存储、病理信息管理、远程会诊等多项业务，全方位服务于病理医生，致力于为业内人士提供全数字化AI病理的整套建设方案

专注AI+病理，致力于通过D-Path系列产品，以专业AI算法，辅助病理诊断及科学研究，成为病理医生的好帮手
打造极致产品，深潜人工智能与数字病理领域，研发深度学习算法，致力打造专属病理诊断的AI系统
用人工智能改变医疗现状，让医生看得更快更准，让病人能够早发现早治疗

三、客户覆盖的行业领域：

医疗领域

四、产品能力：

1.减轻医生工作量
2.提高诊断精度，减少漏诊
3.工作效率翻倍
4.诊断水平迅速提升

五、主要产品：

1.D-CleverEye
2.D-PathAI
3.D-PathPIS

六、产品服务场景覆盖能力：

✨智睫系统D-CleverEye：

实时读取显微镜镜下视野，通过AI算法进行分析，迅速给出提示，几乎无延迟。不打破病理医生日常工作流程的前提下，让AI更有效地帮助医生阅片
适配多品牌显微镜，即装即用
数十万例数据、监督学习算法，保障高精准度标记&计数
语音提示实时播报，所见所闻即所得
远程会诊、轨迹追踪……

✨人工智能辅助病理诊断系统D-PathAI：

上传切片、管理切片、AI分析、报告生成……
实时会诊、双屏对比、模型定制、自训练……

✨病理信息管理系统D-PathPIS：

信息统筹、切片质控、标本追踪、医嘱报告、统计归档、资源管理……
与院内信息系统对接，支持信息一键导入
多账号登录，系统显示模块版权不同，配合使用，方便管理
PC、移动端均可登录
覆盖病理工作全流程
实时远程会诊，无缝搭载AI辅助诊断系统

七、团队实力：

两位创始人均系美国知名大学跨学科教授&博士生导师，在计算机视觉、深度学习和数字病理方向研究15年以上，研发团队超1/3拥有海外博士学位，技术实力有保障。公司总部位于杭州，多地设有分公司，业务范围面向全球。

八、特色优势：

快速精准、适用广泛、分割定位、图像增强

Eau De Cologne

Posted on 2020-03-06 In 乌鸡鲅鱼
秘密深藏于谁的心底，或许我们都知道，只是我们都不说。
夏天来到的时候，春天似乎还未远去。 暮春的花儿大都坚持到了立夏，让这个苍白的世界有了些微的色彩。
日子渐渐变得简单平静下来。
“故事的开始很明媚。”
“你知道么，悲伤的结尾总是要有一个温暖的开局。”
“把美丽的东西毁灭给人看才是悲剧的最高境界。”
“嗯我早就原谅你啦。”
距离那个干净如白纸的年代已经越来越远了。
即便是努力地回忆，留在记忆里的也不过是像电影胶片一闪而过的场景，或是萍水相逢，或是几句苍白的对话。
“人就是太脆弱，所以什么都逃不脱。”
谁都可以记得，当年收到第一封情书时的快乐。只是没有人敢轻易翻动，因为那上面记录了太多的欢笑。 惘然错失，无法不嫉妒，为什么曾经的自己会那样幸福。
“兰，你知道么，这是他送给我的最珍贵的礼物，也是我一生的枷锁。”
或者这便是Lorraine与工藤新一的不同。
总会有什么人什么事触动心中最柔软的部分。 他们像旋转木马在脑海里画圆，它们像海绵吸走全部的想念。
“主角迷失了方向，童话剧终将散场。如果可以相信天堂，那么上帝躲在了什么地方。”
“又或者他并不需要藏躲，因为他就是那个操纵者，导演出此间卑劣的戏码，在幕后注视着台上可悲的小丑们，发出鬼魅而凄厉的嘲笑。”
我以为遗忘之后总是重新开始，但那其实只是另一种意义上的铭记。
时间辗转到这一秒，太多的过往也该就此停驻。 也许以前还会幻想，总有一天可以把这一切忘掉。然而不断出现的波澜就像攀爬的藤蔓，将她们的希求缠绕成无边的绝望。
那么有没有谁可以证明，它曾经真实地存在过。就像荼蘼，花期虽短，盛开的却是无尽芳华。
因为始终放不开那些温暖，所以回忆是最痛苦的祭奠。
“暗夜，月光，钢琴，舞蹈。”
“路西法走到欧若拉身边，他递给她一杯红酒，他对着她绽开鬼魅的笑。”
“是谁还在保持沉默，上帝说错的人不是我。”
“路西法看着欧若拉将杯中的红酒一饮而尽，眸光变得阴冷而凌厉。”
“Don't worry, my love.”
“Everything is fantastic and you will be the next.”
有些事情的改变，需要某种特别的契机。
流水断了线汇聚成时间。
表象的可怕之处在于它用虚假欺瞒了太多，骗过了人的双眼，骗住了人的心。
“世界旋转，转不过宿命的轮回。”
“Death itself is powerless to still a remembered voice or erase a remembered smile.”
“希望你可以幸福。”
可以记得么，还是都忘了。
她怀念的是此间彼间，她怀念的是那个下雨天，她怀念的是他们撑着伞，她怀念的是他把伞倾向她这边。 工藤新一给的幸福如同美丽却短暂的梦，来不及等到天亮，就已经散场。
那么够了吧，可是够了么。
“Thank you, my girl.”
都过去了吧，还过得去么。
回忆之所以珍贵是因为那些短暂的美好，它们一个接着一个连成片段，时常记起而挥之不去。 比如偶尔回眸会看见他干净的侧脸，比如每次分别会听见他极不情愿地说再见，比如他讲起福尔摩斯时总带着神采奕奕的表情，比如他第一次吻她时明明很紧张却还要强装淡定。 时光流逝并没有带走最本质的思念，它们反而越来越深，如侵入骨髓的毒。
“Liar. Murderer. Lucifer.”
“铭记无非有两种方式，或爱至深，或恨至极，因为你不会选择前者，所以我只能让你属于后者。”
“以后要找一个人，替我好好爱你。”
三个小时前，当Lorraine说出这句话的时候，她想到了她的少年，他也曾经这样告诉过她。
可是，再也不会了。

鉴权方式及相关概念

Posted on 2020-02-15 In 前端工程

核心概念

  • 鉴权≠算法/加密≠HTTPS

    鉴权

  • 基础鉴权
  • session/cookie
    优点：较易扩展；简单
    缺点：安全性低；性能低，服务端存储；多服务器同步session困难；跨平台困难
  • JWT（Json Web Token）
    优点：易扩展；支持移动设备；跨应用调用；安全；承载信息丰富；无状态，编码数据
    缺点：刷新与过期处理；Payload不易过大；中间人攻击
  • Oauth
    优点：开放；安全；简单；权限指定
    缺点：需要增加授权服务器；增加网络请求
  • …

算法/加密

  • Base64
  • MD5/SHA-1
  • DES/AES
  • RSA/ECC
  • …

HTTPS

  • SSL
  • HTTP劫持
  • 数据篡改
  • 敏感信息
  • 中间人

JWT工作原理

  • JSON WEB TOKEN
  • 一个JWT由三部分组成：头部(head)，载荷(payload)，签名(signature)
  • 一个好玩的jwt网站

vueBasis

Posted on 2019-12-31 Edited on 2020-02-11 In javascript框架

VUE是什么

  • 构建用户界面的渐进式框架

VUE特点

  • 自底向上
  • 核心库只关注视图层
  • 易上手
  • 便于与第三方库或既有项目整合

兼容性

  • 不支持IE8及以下版本
  • 支持所有兼容ECMAScript 5的浏览器

安装

  • 直接使用<script>引入

Vue会被注册为一个全局变量

  • NPM

  • 命令行工具（CLI）

声明式渲染

Vue.js的核心是：
一个允许采用简洁的模板语法来声明式地将数据渲染进DOM的系统。
看起来与渲染一个字符串模板非常类似，但是Vue在背后做了大量工作。
数据与DOM之间建立关联，所有东西都是响应式的。

  • 文本插值

指令

  • v-bind
    指令绑定元素特性
    在渲染的DOM上应用特殊的响应式行为
  • v-if
    即不仅可以把数据绑定到文本或特性，还可以绑定到DOM结构
  • v-for
    v-for指令可以绑定数组的数据来渲染一个列表等
  • v-on
    可以添加事件监听器，通过它调用在Vue实例中定义的方法
  • v-model
    轻松实现表单输入和应用状态之间的双向绑定

docker

Posted on 2019-12-29 In docker

Docker主要特性:

  • 文件、资源、网络隔离
  • 变更管理、日志记录
  • 写时复制

安装docker

Mac上安装docker

  1. 注册一个docker ID
  2. 下载docker desktop
  3. 下载后打开APP就可以在终端使用docker命令啦

Linux上安装docker

  1. https://docs.docker.com/install/linux/docker-ce/centos/

  2. 安装完成后，
    systemctl start docker
    systemctl status docker

docker的简单命令

  • docker run 命令创建容器
  • -itd 交互式终端后台运行
  • –name 指定容器名称
  • -p指定映射端口 宿主机端口:容器内端口
  • docker rm #dockername 可以删除容器（容器若在运行中则删除不了，必须使用docker stop先停止）

docker镜像加速

在中国区可以使用中国官方镜像加速

修改保存后重启docker以使配置生效。

docker hub

https://hub.docker.com/
