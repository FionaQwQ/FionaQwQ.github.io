docker

Posted on 2019-12-29 In docker

Docker主要特性:

  • 文件、资源、网络隔离
  • 变更管理、日志记录
  • 写时复制

安装docker

Mac上安装docker

  1. 注册一个docker ID
  2. 下载docker desktop
  3. 下载后打开APP就可以在终端使用docker命令啦

Linux上安装docker

  1. https://docs.docker.com/install/linux/docker-ce/centos/

  2. 安装完成后，
    systemctl start docker
    systemctl status docker

docker的简单命令

  • docker run 命令创建容器
  • -itd 交互式终端后台运行
  • –name 指定容器名称
  • -p指定映射端口 宿主机端口:容器内端口
  • docker rm #dockername 可以删除容器（容器若在运行中则删除不了，必须使用docker stop先停止）

docker镜像加速

在中国区可以使用中国官方镜像加速

修改保存后重启docker以使配置生效。

docker hub

https://hub.docker.com/
